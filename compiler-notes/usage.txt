	flags:
	
	-Typestate -typestate -t:	Perform only typechecking.
	-Java14 -java14 -j:		Perform only java1.4 checking.
	Note if you put both of the above flags you get both checks.;

	-Semantic -semantic -s:		Perform only typecheck. No java files are created.
	-Verbose -verbose -v:		Print the steps the typestate typechecker does.


To compile and run:

1) java TypestateMain [main].ses  --> typechecks your code + creates the java files from your code.

2) javac [main].java  -->  the javac compiler will create the .class files

3) java [main]  -->  runs the main method in the [main].class file
